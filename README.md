# Job Landing Page

Bir iş ilanı sayfası - Next.js 15, TypeScript ve Tailwind CSS v4 ile oluşturulmuştur.

Canlı Demo: https://job-landing-pagee.netlify.app

## Özellikler

- Responsive Tasarım - Tüm cihazlarda mükemmel görünüm
- Koyu/Açık Tema - Kullanıcı tercihine göre tema değişikliği
- Smooth Animasyonlar - AOS ile yumuşak geçişler
- High Performance - Next.js 15 ile hızlı yükleme
- SEO Friendly - Arama motorları için optimize edilmiş
- Modern UI/UX - Tailwind CSS v4 ile güzel arayüz

## Teknoloji Stack

- Next.js 15 - React framework
- TypeScript - Tip güvenliği
- Tailwind CSS v4 - CSS framework
- React 18+ - UI library

## Başlangıç Kılavuzu

### 1. Depoyu İndir (Clone)

```bash
git clone https://github.com/yasindabak/Job-Landing.git
cd Job-Landing
```

### 2. Bağımlılıkları Yükle

```bash
npm install
```

veya yarn kullanıyorsanız:

```bash
yarn install
```

### 3. Geliştirme Sunucusunu Başlat

```bash
npm run dev
```

veya

```bash
yarn dev
```

Tarayıcınızda açın: http://localhost:3000

## Üretim İçin Build

Projeyi üretim ortamı için derlemek için:

```bash
npm run build
npm start
```
