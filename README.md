Next.js 15, TypeScript ve Tailwind CSS v4 kullanılarak oluşturulmuş bir İş İlanı (Job Landing) Sayfası. Proje; koyu/açık tema geçişi, AOS ile yumuşak kaydırma animasyonları ve tüm cihazlar için optimize edilmiş profesyonel bir tasarım içerir.Deploy Netlify ile yapılmıştır.

 Projeyi İndirip Çalıştırma

Aşağıdaki adımları izleyerek projeyi kendi bilgisayarınızda çalıştırabilirsiniz.

1. Depoyu İndir (Clone)
git clone https://github.com/yasindabak/Job-Landing.git

Ardından proje klasörüne girin:
cd Job-Landing

2.Bağımlılıkları Yükle

Proje Next.js 15, TypeScript ve Tailwind CSS v4 ile oluşturulmuştur.
npm install

3.Geliştirme Sunucusunu Başlat
npm run dev
